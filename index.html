İcon İsimleri:

SELECTORS
Selects HTML items and perform actions on the selected item (s).
Selectors comment
(*) Selects all items in the document
(this) Selects valid HTML element
("p.into") Selects all <p class="intro"> elements
(div p) Selects all <p> elements inside all <div> elements
(div p:first-child) Selects the first <p> of all <div> elements
("[href]") Selects all elements with an href attribute
HIDE

n1.hide('p') the p element hides.

n1.hide('#p') p id hides the element.

n1.hide('.p') p class hides the element.

SHOW

n1.show('p') the p element shows.

n1.show('#p') p id shows the element.

n1.show('.p') p class shows the element.

n1.toggleShow('[p/#p/.p]') switches between hiding and displaying the selected element.

ADD STYLE

n1.addStyle(selector, 'property', 'value') used in the form of.
Example:

<button onclick="n1.addStyle('.city','background-color','red')">Add Style</button>
ADD CLASS

n1.addClass(selector,'class') used in the form of.
Example:

<button onclick="nj.addClass('h2','marked')">Add Class</button>
FILTERS

n1.filterHTML(selector) used in the form of.
Example:

<button onclick="n1.filterHTML('#id01', 'li')">Sort</button>
<ul id="id01">
<li>Oslo</li>
<li>Stockholm>/li>
<li>Helsinki>/li><li>Berlin>/li>
<li>Rome>/li>
<li>Madrid>/li>
</ul>
SORT

n1.sortHTML(selector) used in the form of.
Example:

<button onclick="n1.sortHTML('#id01', 'li')">Sort</button>
<ul id="id01">
<li>Oslo</li>
<li>Stockholm>/li>
<li>Helsinki>/li><li>Berlin>/li>
<li>Rome>/li>
<li>Madrid>/li>
</ul>
SLIDE SHOWS

n1.slideshow(selector, time (milliseconds)) used in the form of. Makes a slide show.

DISPLAY

n1.displayObject(selector) used in the form of. Simply add parentheses to any HTML item to take part in your data.
