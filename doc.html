CSS

prefix: nc Add into the class tag

COLOR

Can give color to the text.
Example:

<div class="nc-text-cyan">Lorem ipsum dolor sit amet.</div>

Can give color to the background.
Example:

<div class="nc-black">Lorem ipsum dolor sit amet.</div>

Can give color to the border.
Example:

<div class="nc-boreder-cyan">Lorem ipsum dolor sit amet.</div>

Can give color to the hover.
Example:

<div class="nc-hover-cyan">Lorem ipsum dolor sit amet.</div>
<div class="nc-hover-text-cyan">Lorem ipsum dolor sit amet.</div>

CONTAINER

nc-container It is the perfect classroom to use all HTML container elements

<div class="nc-container nc-black">Lorem ipsum dolor sit amet.</div>
PANEL

nc-panel Inserts 16px padding in all html element.
Example:

<div class="nc-panel nc-black">Lorem ipsum dolor sit amet.</div>

Can be used within citations.
Example:

<div class="nc-panel nc-leftbar nc-sand nc-xxlarge nc-serif">Lorem ipsum dolor sit amet.</div>
BORDER

nc-border Provides the following border classes:

Syntax comment
nc-border-top Adds top border to an item
nc-border-right Adds right border to an item
nc-border-left Adds left border to an item
nc-border-bottom Adds bottom border to an item
nc-border-0 erases the border of an item
nc-hover-border Adds hover border to an item
nc-topbar Adds a thick top border to an item
nc-rightbar Adds a thick right border to an item
nc-leftbar Adds a thick left border to an item
nc-bottombar Adds a thick bottom border to an item

TEXT

nc-left-align and nc-right-align classes are used to align the text. The nc-wide class specifies a wider text. The nc-opacity class is designed to work with all colors, changes the text opacity.

ROUND

nc-roundAdds rounded corners to any HTML element.

Syntax comment
nc-round Adds a 4-pixel rounded item
nc-round-small Adds a 2-pixel rounded item
nc-round-medium Adds a 4-pixel rounded item
nc-round-large Adds a 8-pixel rounded item
nc-round-xlarge Adds a 16-pixel rounded item
nc-round-xxlarge Adds a 32-pixel rounded item
MARGIN

nc-margin class adds a 16 pixel margin to all edges of an item. You can set the margin by adding space to the top, right, left, bottom.

PADDING

nc-padding classes add the upper and lower padding to any HTML. You should use Small, Large, 16, 24, 32, 48, 64 to size.



JS

prefix: nj

SELECTORS
Selects HTML items and perform actions on the selected item (s).
Selectors comment
(*) Selects all items in the document
(this) Selects valid HTML element
("p.into") Selects all <p class="intro"> elements
(div p) Selects all <p> elements inside all <div> elements
(div p:first-child) Selects the first <p> of all <div> elements
("[href]") Selects all elements with an href attribute
HIDE

nj.hide('p') the p element hides.

nj.hide('#p') p id hides the element.

nj.hide('.p') p class hides the element.

SHOW

nj.show('p') the p element shows.

nj.show('#p') p id shows the element.

nj.show('.p') p class shows the element.

nj.toggleShow('[p/#p/.p]') switches between hiding and displaying the selected element.

ADD STYLE

nj.addStyle(selector,'property','value') used in the form of.
Example:

<button onclick="nj.addStyle('#text','background-color','red')">Add Style</button>
ADD CLASS

nj.addClass(selector,'class') used in the form of.
Example:

<button onclick="nj.addClass('h2','marked')">Add Style</button>
FILTERS

SORT

nj.sortHTML(selector) used in the form of.

SLIDE SHOWS

nj.slideshow(selector, time (milliseconds)) used in the form of. Makes a slide show.

nj.displayObject(selector) used in the form of. Simply add parentheses to any HTML item to take part in your data.
